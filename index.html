New Venue Openings Link Builder

Build a personalised or filtered link to share with contacts.

1

Personalise for a contact

Adding a contact ID personalises the page for that specific contact. It filters venues to their preferred cities and event types, and shows your name and meeting link on the CTA. This is the recommended approach for outreach emails.

Find this in the HubSpot URL when viewing the contact: app.hubspot.com/contacts/.../contact/12345
Tip: If you add a Contact ID, you usually don't need to set any filters below — the page will automatically show venues matching the contact's preferences in HubSpot.
2

Filter venues optional

Use these to narrow the results. Filters override the contact's preferences if a Contact ID is set — so only add them if the contact asked about something specific (e.g. "show me new conference venues in Manchester"). Without a Contact ID, the page defaults to London.

Hold Cmd/Ctrl to select multiple. Only venues in these cities will show.
3

Display options optional

Group venues together by city, area, or type. This adds a dot navigation on the right side with group labels. Useful when showing venues across multiple cities or types.

When to use: "Group by City" is great when sharing venues across multiple cities (e.g. London + Manchester). "Group by Area" works well within a single city to organise by neighbourhood. "Group by Venue Type" helps when showing a mix of hotels, restaurants, etc.
